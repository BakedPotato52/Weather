Features

1. Current Weather Information:

Display real-time weather conditions such as temperature, humidity, wind speed, and cloud cover for a specific location.

2. Location-based Forecast:

Allow users to input or select their location to receive a personalized weather forecast.

3. Multiple Locations:

Enable users to add and track weather information for multiple locations, making it convenient for travelers or users interested in different places.

4. User Preferences:

Allow users to customize their weather preferences, such as choosing a default location, setting notification preferences, or selecting their preferred units of measurement.

5. Responsive Design:

Ensure that the application is accessible and user-friendly on various devices, including desktops, tablets, and smartphones.
